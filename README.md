# Discord Bots

Prototipos de bots de **Discord** en JavaScript (Node.js). Proyectos de práctica para explorar comandos, eventos y la API de Discord.

## Bots

Cada carpeta (`Bot 1`, `Bot 2`, `Bot 3`) contiene un prototipo independiente.

## Uso

Cada bot requiere su propio token de Discord:

```bash
npm install discord.js
node "Cheese bot.js"
```

> ⚠️ Nunca subas tu token de Discord al repositorio. Usa variables de entorno.
